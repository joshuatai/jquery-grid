• Use actionItemDisable method after initialization.

    • Description: You can disable or enable any action button after initialized tmGrid instance.
    • Arguments: item- while action item you want to disable.
    • Code examples:

          var item= $('#tmGrid').tmGrid('option', 'actionItems')[0].element;
          $('#tmGrid').tmGrid('actionItemDisable', item);

  • Use option(getter) method after initialization

    • Description: Get any tmButton option.
    • Code examples:

      var value= $('#tmButton').tmButton('option', 'optionName');
