• Initial with type option

    • Attribute Key: type
    • Type: String
    • Default: get
    • Description: You can set a get or post method type to reqest server side.
    •  
    • Code examples: Initialize a tmGrid with the type option specified.

          $('#tmGrid').tmGrid(
              {
                  'dataProvider': 'handler/handler.php',
                  'columns': columnsSetting
                  'Type': 'get'
              }
          );

      Get or set the type option, after init.
      //getter
          var type= $('#tmGrid').tmGrid('option', 'type');

      //setter
          $('#tmGrid').tmGrid('option', 'type', newtype);
