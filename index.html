Documentation of tmGrid Component

Overview:

  • tmGrid enhances standard form elements as table with visual standard, appropiate search function, sort behavior, pagination, and action button.
  • When initialing a tmGrid component, please use table element to implement.
  • A call to $('#tmGrid').tmGrid({ dataProvider:..., columns:{...}}) will initialize a tmGrid instance.

Required:

  • jquery lib
  • jquery UI
  • ui.tmGrid.js
  • ui.tmGrid.css
  • ui.tmAJAX.js

Optional:

  • dateFormat.js
  • ui.tmDropDownList.js
  • ui.tmDropDownList.css
  • ui.tmPopup.js
  • ui.tmPopup.css

  • Initial with dataProvider and column option

    Top
    • Attribute Key: dataProvider
    • Type: String | Array
    • Default: null
    • Description: It provides a data source for showing a tmGrid. It cannot be empty when initialising (first creating) the tmGrid.
    •  
    • Attribute Key: columns
    • Type: Array
    • Default: null
    • Description: It provides a group of column setting for showing the columns header. It cannot be empty when initialising (first creating) the tmGrid.
    •  
    • Dependency: It must be implemented with dataProvider and columns at first.
    • Code examples: Initialize a tmGrid with the dataProvider and columns option specified.
