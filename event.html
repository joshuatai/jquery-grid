• Initial with pageChange event.

    • Type: Function
    • Default: null
    • Description: You can assign an event handler with page changing behavior of this tmGrid instance.
    • Code examples:

      Initialize a tmGrid with the pageChange option specified.
          $('#tmGrid').tmGrid({
              pageChange: function(event, self){
                  alert("Page change event has been triggered!");
              }
          });
      Get or set the pageChange option, after init.
      //getter
          var pageChange= $('#tmGrid').tmGrid('option', 'pageChange');
      //setter
          $('#tmGrid').tmGrid('option', 'pageChange', function(event, self){ do the other thing…. });

  • Initial with rowSelect event.

    • Type: Function
    • Default: null
    • Description: You can assign an event handler for row selection of this tmGrid instance.
    • Dependency: Option selectable must to be true.
